# Invoice Processing API

A RESTful Web API built using ASP.NET Core, Entity Framework Core, and SQL Server.

## Features

 Create Invoice (POST)
 Get All Invoices (GET)
 Get Invoice By Id (GET)
 Update Invoice (PUT)
 Delete Invoice (DELETE)
 Input Validation
 Swagger API Documentation

## Technologies Used

 ASP.NET Core Web API
 Entity Framework Core
 SQL Server
 SwaggerOpenAPI
 C#

## Project Purpose

This project was developed to learn and demonstrate CRUD operations, Entity Framework Core, Dependency Injection, API validation, and SQL Server integration.
